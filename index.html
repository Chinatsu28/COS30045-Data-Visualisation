Choropleth Analysis

How to use this visualisation

Users can select parameters like "inflow" or "outflow" to view immigration data from different perspectives. They can explore the treemap visualization by clicking on regions or countries, where the size of each rectangle represents immigration numbers and colors indicate continents. Interacting with the choropleth map involves hovering over regions or prefectures to see detailed immigration statistics, with color intensity reflecting the magnitude of immigration. Users can navigate the tornado chart to visualize population changes by age group, clicking on bars to switch between inflow and outflow data for different regions. Providing a feedback option allows users to share thoughts on usability and clarity, while including a reset or clear selection feature enables exploration of different datasets. Accessible documentation or tooltips within the interface further aids users in understanding how to interact effectively with each component.
